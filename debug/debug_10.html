5. 출력문과 내장 함수

Python에서 데이터를 출력하거나 다양한 작업을 수행하기 위해 내장 함수를 사용할 수 있다.
- `print()`: 출력 함수
- `len()`, `dir()`, `type()`: 다양한 내장 함수

 7 . 문자열 처리

문자열은 다양한 방법으로 처리할 수 있다.
- 인덱싱: `a[0]`
- 슬라이싱: `a[1:3]`
- 길이: `len(a)`

 

 

둘째날

8. 조건문

조건문은 특정 조건에 따라 코드 블록을 실행할 수 있게 해준다.

대입 연산자: = (변수에 값을 할당한다)
값 일치: == (값이 같은지 비교한다)
타입 일치: === (타입까지 일치하는지 비교한다)
부정: ! 또는 not (부정 연산자를 사용한다)
값이 일치하지 않음: !=
비교 연산자: <=, >=
문자열 비교 연산에서는 'a' < 'b'와 같이 알파벳 순서대로 비교한다.

9. 논리 연산자

not: 단항 연산자로 조건을 부정한다.
and: 이항 연산자로 두 조건이 모두 참일 때 참이 된다.
or: 이항 연산자로 하나의 조건이라도 참이면 참이 된다.

 


10. 리스트 (List)

리스트는 여러 요소를 저장할 수 있는 가변(mutable) 자료형이다.

리스트 생성: [요소, 요소, 요소]
반복문과 결합: 이터러블 객체로서 반복문과 함께 사용할 수 있다.
다중 리스트: [[요소, 123], [123, "abc"], "def"]
빈 리스트: [x for x in ... if 조건]
중복 허용: ["사과", "사과", "딸기"]

11. 반복문

for 반복문

 


시작과 끝을 알고 있을 때 사용한다.

 

 

while 반복문

시작과 끝을 모를 때 사용한다.
시작 값이 반복문 밖에 있는지 안에 있는지 차이가 있다.

1
2
3

5
count = 0
while count < 5:
    print(count)

 cs


참인 동안 무한히 반복한다.

continue 문을 사용하여 현재 반복을 생략하고 다음 반복으로 넘어간다.


12. 유용한 내장 함수

min(): 최소값을 반환한다.
max(): 최대값을 반환한다.
sum(): 합계를 계산한다.
reversed(): 역순으로 반복한다.
enumerate(): 현재 인덱스와 요소 값을 튜플로 반환한다.
items(): 딕셔너리의 키와 값을 튜플로 반환한다.

15. 제너레이터

yield를 사용하여 함수 내부의 코드 실행을 중지한다.
next()를 사용하여 제너레이터의 다음 값을 실행한다.

1
2

def my_generator():
    yield 1
    yield 2
    yield 3
 
gen = my_generator()
print(next(gen))  # 출력: 1
print(next(gen))  # 출력: 2

16. 스택과 힙

스택: 순서가 있으며 빠른 접근이 가능하다. 원시 자료형을 차곡차곡 저장한다.
힙: 자유롭게 접근 가능하지만 상대적으로 느리다. 데이터를 저장하는 창고와 같다.
