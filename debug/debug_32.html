KPT

  • Keep : 현재 만족하고 있는 부분, 계속 이어갔으면 하는 부분
  • Problem : 불편하게 느끼는 부분, 개선이 필요하다고 생각되는 부분
  • Try : Problem에 대한 해결책, 실행 가능한 것

KPT 방식으로 1주차 회고를 간단하게 적어보고자 한다.

 

Keep

가장 좋은 점을 말하자면 대면으로 교육이 진행된다는 점이다.

억지로라도 이른 아침에 일어나고, 규칙적인 생활이 시작되었다는 점이 매우 만족스럽다. 교육 시설은 깔끔하고 새로 지은 느낌이 나서 좋다. 건물 꼭대기 층이여서 그런지 뷰도 괜찮다.

강사님의 강의력도 좋으시다.
함수, 클래스와 같은 중요한 문법을 정말 이해하기 쉽게 설명해주신다. 앞으로 진행될 본격적인 데이터 분석과 머신러닝, 딥러닝 관련 강의도 기대가 된다. 강사님께서 중간중간 해주시는 소소한 이야기도 뻘하게 재밌다.

다양한 분들과의 만남

2일동안 10명정도의 수강생 분들과 이야기를 해봤던 것 같다. 나이대도 다양했고 무엇보다 본인이 가장 나이가 적다는 사실에 살짝 당황했다. 전공도 다양하고 각자 상황도 달랐기 때문에 대화를 하면서 재밌었다. 전공 공부라는 울타리 안에 있다가 더 넓은 경험을 하게 된 것 같아서 보람찼다.

 

Problem

출퇴근 시간의 1호선이 가장 큰 문제다.

사람도 많고 무엇보다 배차 간격이 불규칙적이여서 정신을 똑바로 차리고 있어야 한다. 점점 노하우가 생길 수 있도록 노력해야겠다.

이번 1주차는 파이썬 기본 문법에 대한 학습이 이루어졌다.
파이썬이 가지고 있는 고유한 문법에 익숙해질 필요가 있어 보인다. 예를 들어, comprehension 문법, 함수 정의 방법, 람다식 등등. 앞으로 주력 언어가 파이썬이 될 수 있도록!

 

Try

이번주는 안전하게 8시30분 등원을 목표로 둘 생각이다.
열차 한번 놓치면 일단 20분은 늦는다고 생각해야 한다.

파이썬을 가지고 이전에 풀었던 프로그래머스 코딩테스트 문제, 백준 문제 등을 다시 풀어볼 예정이다.
욕심내지 않고 하루에 1문제는 꼭 풀것이다.

안그래도 친화력이 부족한데 3일이나 늦게 합류해서 사람들과 친해질 시간이 부족했다.

학습도 학습이지만, 엠비티아이 앞자리가 E가 되도록 한걸음 한걸음 노력해야겠다.
