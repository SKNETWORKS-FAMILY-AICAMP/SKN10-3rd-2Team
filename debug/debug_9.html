FACTS

 Python 프리코스 수강

 프리코스는 정규 수업을 듣기 전에 미리 공부할 수 있도록 Playdata에서 제공해주는 온라인 강의입니다. 정규 수업 시작일( 02.26. 수 ) 전에 프리코스를 통해 파이썬 기본 문법과 사용법을 배웠습니다. 덕분에 정규 수업 때 수업 진도를 편하게 따라갈 수 있었습니다. 파이썬 이외에도 자바와 깃 코스도 있는데 남는 시간에 들어보려고 합니다.

프로그래머스 코딩 테스트 응시( PCCE )

 캠프에서 프로그래머스 인증 시험 응시를 지원해 주는 것은 저에게 매우 반가운 소식이었습니다. 코딩 테스트 경험이 없고, 어떻게 준비해야 할지 막막한 참이었는데 이번 기회에 코딩 테스트 준비를 시작해야겠다고 생각했습니다. 마침 2월 코딩테스트가 1일차( 02.26 )에 있어서 수업을 마친 후 응시하였습니다. 시험은 정기적으로 진행될 예정이라고 합니다. 코딩 테스트 실력을 늘릴 수 있는 좋은 기회인 것 같습니다.

정규 수업 수강

 정규 수업은 2025년 2월 26일( 수 )에 시작되었습니다. 1주차는 3일간 수업을 들었는데 짧은 기간 동안 파이썬에 대한 기본적인 사용법을 배울 수 있었습니다. 가장 인상 깊었던 것은 파이썬을 공부할 때 GPT를 적극 활용하라는 강사님의 말씀이었습니다. 저도 파이썬 뿐만 아니라 다른 언어를 공부할 때 GPT를 활용하는 편이었는데, 단순히 개념만 공부할 것이 아니라 GPT로 관련 예제를 여러 개 만들어 보고 많이 활용해보는 방식으로 공부를 하는게 좋다는 말에 크게 공감하였습니다. 앞으로의 학습에 이 방식을 적용해 보려고 합니다.

FINDINGS

Python, Jupyter, Colab 사용법
 새로 배운 점은 파이썬 코드를 실행할 수 있는 컴퓨팅 노트북으로 google의 Colab이 있다는 것입니다. 주로 사용했던 것은 Jupyter와 다르게 소스파일이 구글 드라이브에 저장이 되고, 실행도 구글이 제공해주는 가상의 컴퓨터 공간에서 돌아갑니다. 웬만한 라이브러리들이 최신 버전으로 설치되어 있고 코드 공유가 쉽다는 장점이 있습니다. 하지만 가상 환경을 만들거나 로컬 자원을 활용하기 어렵기 때문에 상황에 맞춰 Jupyter와 Colab을 사용해야 합니다.
Python 기초 문법
