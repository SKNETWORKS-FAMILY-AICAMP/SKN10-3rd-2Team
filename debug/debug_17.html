Keep  (현재 만족하고 있는 부분, 계속 이어갔으면 하는 부분)

  1. 바뀌었던 생활 패턴을 다시 맞추기 위해 OT 당일에 밤을 새고 다음날 정상적인 컨디션으로 강의를 들을 수 있었다. 기간 동안 규칙적인 생활을 잘 유지하자.
  2. OT가 끝나고 헬스장을 등록했고 아무리 힘들어도 적어도 2일에 1번은 운동을 할 수 있도록 하자.

Problem  (불편하게 느끼는 부분, 개선이 필요하다고 생각되는 부분)

  1. Mac을 사용 중인데, 과정에서는 windows를 사용해서 나중에 복습하며 정리할 때 불편함이 있을 것 같다.
  2. 진행 중이던 코딩테스트 스터디와 앞으로 병행하게 될텐데, 전보다 알고리즘 문제 풀이할 시간이 꽤 부족하다.

Try (Problem에 대한 해결책, 실행 가능한 것)

  1. 대여한 노트북으로 당일에 강의를 마친 후 복습하고 블로그에 글을 정리하고 가는 것이 효율적일 것 같다.
  2. 피곤하더라도 주말에 몰아서 풀지 말고 하루에 한 문제씩 해결할 수 있도록 하자.
