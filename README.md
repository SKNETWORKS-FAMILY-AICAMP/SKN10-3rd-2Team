# SK네트웍스 Family AI 캠프 10기 - 부트캠프 RAG 챗봇

부트캠프 관련 질문에 답변해주는 RAG(Retrieval-Augmented Generation) 기반 챗봇입니다.

## 설치 방법

### 1. UV를 사용한 설치 (권장)

UV는 빠른 Python 패키지 설치 도구입니다. 다음 단계를 따라 설치하세요:

```bash
# UV 설치
pip install uv

# 코드 클론
git clone https://github.com/your-repo/SKN10-3rd-2Team.git
cd SKN10-3rd-2Team

# 설치 스크립트 실행
python setup.py
```

또는 직접 uv로 설치:

```bash
uv pip install -r requirements.txt
```

### 2. 기존 pip를 사용한 설치

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run Home.py
```

## 기술 스택

### 프론트엔드
- **Streamlit**: 대화형 웹 인터페이스 구현
- **HTML/CSS**: 웹 페이지 스타일링 및 사용자 경험 최적화

- **Python**: 핵심 비즈니스 로직 구현

### AI/ML
- **RAG(Retrieval-Augmented Generation)**: 정보 검색 및 응답 생성 아키텍처
- **Ollama**: 로컬 LLM 실행 환경
- **Gemma 2B**: 응답 생성을 위한 기반 모델
- **BM25**: 텍스트 검색 및 순위 매김 알고리즘

### 데이터 처리
- **Pandas**: 데이터 전처리 및 분석
- **CSV/JSON**: 데이터 저장 및 관리

## 진행 과정 중 프로그램 개선 노력

### RAG 시스템 개선
1. **검색 정확도 향상**
   - 임계값 조정을 통한 검색 결과 필터링 최적화 (-1500 음수 값으로 조정)
   - BM25 알고리즘 도입으로 텍스트 기반 재순위화 구현
   - 자카드 유사도와 코사인 유사도를 결합한 하이브리드 유사도 계산 구현

2. **대화 맥락 개선**
   - 이전 질문을 활용한 컨텍스트 강화 메커니즘 도입
   - 최근 2턴(4개 메시지)을 활용한 연속 대화 처리 개선
   - 응답 프롬프트 최적화를 통한 관련성 높은 답변 생성

3. **성능 최적화**
   - 벡터 저장소 캐싱 구현으로 검색 속도 향상
   - LLM 파라미터 최적화 (컨텍스트 창 확장, 온도 조정)
